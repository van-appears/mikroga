# Mikroga

An attempt to learn the basics of [Love 2d](https://love2d.org/) by creating a simplistic version of Ikaruga.

This is using the following tools
* [VSCode template for Love](https://github.com/Keyslam/LOVE-VSCode-Game-Template)
* [Classic Object module](https://github.com/rxi/classic)

The changelog lists the difference between each iteration. Each iteration is tagged, so you can compare versions using Github's compare functionality e.g. the change from 0.0.1 to 0.0.2 can be seen with URL https://github.com/van-appears/mikroga/compare/0.0.1...0.0.2 (FYI, this may occassionaly include small formatting changes as well as functional changes, and possibly additions to things like this README).

Note that this is mostly a coding exercise. Real games require much more play testing and level design than I am intending to put into this.

## Notes

As Love2d itself seems to use camelCase in its APIs, so will this project (excepting global constants in upper plus snakecase)
